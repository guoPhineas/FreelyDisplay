## 功能

- 在Mac上创建虚拟显示器
- 通过网络共享显示器或窗口
- 可监听多个屏幕
- 图形化操作，简单明了



## 作用

- 解决显示器不够的问题
- 用于在线会议等共享屏幕时创建虚拟显示器以避免屏幕杂乱
- 在线会议等将屏幕通过网络实时共享给参会者
- 无物理显示器，但手边有其他空闲设备
